# kio.dev

Welcome to the monorepo for my devblog / portfolio!

🌐 **Live Site:** [kio.dev](https://kio.dev/)

## 📚 Tech Stack
### Front-end
- SvelteKit
- TypeScript
- TailwindCSS

### Deployment
- Netlify - Hosting and continuous deployment

## 🚀 Development
### Prerequisites

- Node.js ~23
- PNPM ~9.15

### Setup

```bash
git clone git@github.com:kiosion/kio.dev.git
cd kio.dev
pnpm dev
```
