# Project setup

I'm using Vue 3, Tailwind.css, and TypeScript.

#### Install deps:

`npm install`

#### Compiles and hot-reloads for development:

`npm run serve`

#### Compiles and minifies for production:

`npm run build`

#### Lints and fixes files

`npm run lint`
